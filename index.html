JFiddle - Java Fiddle
JFiddle is an online platform to fulfill all the Java needs at a single place.
It's a technology that uses CORS(Cross Origin Resource Sharing) & cloud Computing to implement a set of modules which were difficult to implement previously.
Following features are a part of Jfiddle.
1. Java Compiler- A modern Compiler to code and run java programmes online along with modern features like one click code creator and text highlighter.
2. Fiddle Rank - A modern coding contest which tests your coding skills against set of questions.
3. Java API - A 3 step process to integrate your own java compiler into your own .html site. Plus create your own coding contest by using the API.
4. Java modules - Use Java to add functionalities to your static website by just copying the pre built modules.
