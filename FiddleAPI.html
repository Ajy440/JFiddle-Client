JFiddle API

Add Java API to your website in 3 Simple Steps.

1. Copy the Modernizer and Jquery in your head tag.




Copy the Modernizer and Jquery Css inside your <head> tag. You can also import these scripts from external sources.


Scripts

file_copy Copy
<script src="https://cdnjs.cloudflare.com/ajax/libs/modernizr/2.8.3/modernizr.min.js"></script>

<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.4.1/jquery.min.js"></script>

2. Copy the Form inside your Body tag.




Copy the Html form inside your <body> tag. You can also use data from existing forms by modifying the code.


Forms

file_copy Copy
<form id="form">
	<textarea name="code" rows="10" cols="30" id="code">class Main{
	public static void main(String [] abc){
	System.out.println("hello java"); } }
	</textarea>
	<textarea name="input" rows="10" cols="30" id="input"></textarea>
	<textarea name="output" rows="10" cols="30" id="output"></textarea>
	<input type="button" value="submit" id="runcode">
</form>

3. Copy the Javascript in your html page..




Copy the JavaScript form inside your <body> tag just after the form you pasted. You can also fix the location of js as per your preferenece.


Javascript

file_copy Copy
<script type="text/javascript">

  $(document).ready(function(){

     $("#runcode").click(function(){

           $("#output").html("Loading ......");
   });

  });
//wait for page load to initialize script
$(document).ready(function(){
    //listen for form submission
    $('#runcode').on('click', function(e){
     
      e.preventDefault();

      // AJAX
      $.ajax({
            type: "POST",
            
            cache: false, 
            url: "https://565656.ngrok.io/API/API_compilers/java.php",
            datatype: "html",
            data: $('form').serialize(), 
            success: function(result) { 

            $("#output").html(result);
    }
              
        });
    });
  });
</script>
